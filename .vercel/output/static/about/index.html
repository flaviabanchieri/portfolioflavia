Sobre mim

Flavia Banchieri Roca, Desenvolvedora de Software Pleno com foco em sistemas corporativos, APIs REST e arquitetura de software.

Experiência

Sou Desenvolvedora de Software Pleno, especializada no desenvolvimento de aplicações web, APIs REST e sistemas corporativos utilizando o ecossistema .NET e Angular. Atualmente faço parte da equipe da Secretaria de Estado de Tecnologia da Informação de Mato Grosso do Sul (SETDIG/MS), onde participo da manutenção e evolução de diversos sistemas estratégicos utilizados pela administração pública estadual.

Minha experiência abrange todas as etapas do desenvolvimento de software, desde o levantamento de requisitos e modelagem de banco de dados até a implementação de novas funcionalidades, integrações entre sistemas, otimização de desempenho e sustentação de aplicações em produção.

Ao longo da minha carreira, atuei em projetos dos setores público e privado, desenvolvendo soluções para gestão pública, saúde, educação, agronegócio, varejo, atendimento e tecnologia jurídica. Tenho facilidade para trabalhar em equipes multidisciplinares, utilizando metodologias ágeis e aplicando boas práticas de engenharia de software para construir aplicações escaláveis, seguras e de fácil manutenção.

Sou apaixonada por arquitetura de software e busco constantemente aprimorar meus conhecimentos em tecnologias modernas, design de sistemas, microsserviços e padrões de desenvolvimento.

Stack Tecnológica

Backend: C#, .NET Framework, .NET Core, ASP.NET MVC, ASP.NET Core, Web API, Entity Framework, Entity Framework Core, Dapper, JWT, SignalR, AutoMapper, FluentValidation, Polly e MediatR.

Frontend: Angular, AngularJS, TypeScript, JavaScript, HTML5, CSS3, Bootstrap, Razor e jQuery.

Banco de dados: PostgreSQL, SQL Server, Oracle e MySQL.

Arquitetura: Clean Architecture, SOLID, Clean Code, DDD, Repository Pattern, Unit of Work, Dependency Injection, REST APIs e Microsserviços.

DevOps: Docker, Git, Azure DevOps, GitHub, Nginx, CI/CD, Scrum, Kanban, Postman e Swagger.
